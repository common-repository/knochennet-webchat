Fügt den Webchat vom {IRC}knochennet in euer Wordpress ein.

== Description ==
Fügt den Webchat vom {IRC}knochennet in euer Wordpress ein. Dazu muss nur ein Shortcode in die jeweilige Seite/Artikel. <code>&#91;knochennet_webchat&#93;</code> oder <code>&#91;knochennet_webchat channel="channel"&#93;</code> wenn ein bestimmter Channel im {IRC}knochennet betreten werden soll.

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.
